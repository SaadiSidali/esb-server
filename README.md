# Server for job-hiring-thingy-idk

## Install dependencies and run it

```sh
npm i
npm run start:dev
```

GraphQL endpoint should be available at 

[http://localhost:3000/graphql](http://localhost:3000/graphql)

To set db options check  **/src/.env.example** 

```ts
DB_HOST=
DB_PORT=
DB_USERNAME=
DB_PASSWORD=
DB_DATABASE=

JWT_SECRET=
JWT_EXPIRESIN=
```
